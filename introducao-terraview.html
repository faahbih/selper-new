Introdução ao TerraView (Geração 5)

Objetivo

Dar ao usuário condições de operação do TerraView.

Pré-requisito

Para melhor aproveitamento desse curso, incluímos como pré-requisitos conhecimentos teóricos ou práticos no campo de geoprocessamento (tipos de mapas, representações, cartografia básica e funções básicas de um SIG). Não haverá exigências quanto estes pré-requisitos, portanto, se necessário acesse as referências abaixo para uma leitura pré-curso.

  • Geoprocessamento e Cartografia
  • Sensoriamento Remoto
*Leituras adicionais veja livro on-line de Ciência da Geoinformação.

Destaques

Através de exemplos práticos de manipulação e análise de dados geográficos, o aluno utilizará maioria das funções oferecidas pelo software.

A quem se destina

Profissionais das áreas de recursos naturais (cartografia, geografia, geologia, agronomia, floresta, ecologia etc.) e técnicos que necessitem utilizar o sistema para tratar dados espaciais.


Conteúdo do Curso

(1) Introdução ao TerraView

  • Sobre o TerraView
  • Modelagem de Dados - Projetos
  • Principais Ferramentas

(2) Entrada de Dados

  • Acesso a dados vetoriais
  • Acesso a tados tabulares externas com e sem geometria
  • Acesso a dados matriciais
  • Acesso e intercâmbio de dados geográficos no banco PostgreSQL + PostGIS
  • Propriedades das camadas

(3) Consultas de Objetos e Análises Básicas

  • Consulta por Atributos
  • Consulta Espacial

(4) Operações Geográficas

  • Dissolve
  • Merge
  • Intersecção
  • Diferença
  • União
  • Criação de Buffers

(5) Edição Vetorial

  • Ferramentas de Edição

(6) Análise Espacial

  • Mapa de Kernel

(7) Processamento Digital de Imagens

  • Fundamentos de Sensoriamento Remoto
  • Contraste
  • Filtragem
  • Operações Artmética

Material Didático

  • Apostila impressa para os exercícios práticos;
  • DVD com o material didático em PDF, os bancos de dados que serão manipulados no curso e a versão atual do SPRING.
    • Observação: Pode haver sobreposição de conteúdo entre os cursos por não haver pré-requisito para realizá-los.

Certificado

30 dias após o curso o aluno recebe o certificado da SELPER.
