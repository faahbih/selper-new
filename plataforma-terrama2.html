Plataforma de Monitoramento Ambiental TerraMA²

Sobre o curso

TerraMA2 - Plataforma de desenvolvimento de sistemas de Monitoramento, Análise e Alerta. Esta plataforma permite coletar e cruzar em tempo real os dados geo-ambientais com mapas de riscos e vulnerabilidade ambientais, com o objetivo de gerar alertas para diversas aplicações, inclusive desastres naturais. É um produto de software livre, desenvolvido utilizando-se como plataforma de desenvolvimento a biblioteca de TerraLib. Ele é baseado em uma arquitetura de serviços, aberta, que provê a infraestrutura tecnológica necessária ao desenvolvimento de sistemas

Objetivo

Capacitar equipe técnica a elaborar, editar e construir base de dados para operação da plataforma TerraMA2.

Pré-requisito

Para melhor aproveitamento desse curso, incluímos como pré-requisitos conhecimentos teóricos ou práticos no campo de geoprocessamento (tipos de mapas, representações, cartografia básica e funções básicas de um SIG). Não haverá exigências quanto estes pré-requisitos, portanto, se necessário acesse as referências abaixo para uma leitura pré-curso.

  • Geoprocessamento e Cartografia
  • Sensoriamento Remoto
*Leituras adicionais veja livro on-line de Ciência da Geoinformação.

Destaques

Através de exemplos práticos de construção de sistemas de monitoramento, análise e alerta o aluno utilizará a maioria das funcionalidades oferecidas pela plataforma TerraMA2.

A quem se destina

Profissionais das áreas de recursos naturais (geografia, geologia, agronomia, floresta, ecologia etc.) e profissionais da defesa civil, bombeiros e concessionárias que necessitem monitorar, em tempo real, dados geo-ambientais (meteorológicos, climáticos, geo-técnicos, etc) em suas aplicações.

Conteúdo do Curso


(1) Apresentação geral do TerraMA²
(2) Instalação e configuração do TerraMA²

  • Versão Windows com máquina virtual LINUX
  • Versão LINUX Ubuntu

(3) Bases de daods para um sistema de monitoramento

  • Dados Hidrológicos
  • Dados Meteorológicos
  • Dados Ambientais

(4) Banco de Dados e TerraView

  • Criação de BDG – PostgreSQL + PostGIS
  • Uso do TerraView e preparação dos dados

(5) Módulo de Administração

  • Configuração do Banco de Dados
  • Configuração e Execução dos Serviços

(6) Módulo de Configuração

  • Coleta de dados ambientais
  • Configuração de dados estáticos
  • Configuração de análises
  • Configuração de usuários

(7) Módulo de Monitoramento WEB

  • Clientes dos Alertas
  • Saídas do sistema
  • Aplicativo WEB
  • Notificação via email
