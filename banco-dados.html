Banco de Dados Geográficos

Objetivo

Apresentar conceitos básicos sobre Banco de Dados, com seus diferentes modelos e arquiteturas, incluindo aplicações em Sistema de Informações Geográficas.

Destaques

Modelagem de Banco de Dados Geográficos e consulta espacial.

A quem se destina

Usuários de Geoprocessamento, projetistas de aplicações que usam Banco de Dados e programadores de aplicações.

Conteúdo do Curso (Teórico)


(1) Sistemas de Bancos de Dados

  • Introdução a Sistemas de Bancos de Dados
  • Introdução a SQL
  • PostgreSQL

(2) Modelo de Dados em Geoprocessamento

  • O que é um SIG e sua estrutura
  • Quatro universos da representação do dado geográfico,
  • Noções de cartografia,
  • Arquitetura de BDGeográficos (Dual e Integrada).

(3) Modelagem de Banco de Dados Geográficos

  • OMT-G
  • UML GeoFrame
  • Ferramenta CASE

(4) Sistemas de Bancos de Dados Espacial

  • Introdução a Sistemas de Bancos de Dados Espacial
  • PostGIS

(5) Web Services Geográficos

  • Padrões OGC Web Services (OWS)
  • Web Map Service (WMS)
  • Web Feature Service (WFS)
  • Simple Features Specification For SQL (SFS-SQL)

(6) Métodos de Indexação Espacial e Operadores Topológicos - SFS

  • Exemplos de aplicações em BDG (Projeto PROARCO, DETER, SIGMUN, SISMADEN)



Conteúdo do Curso (Prático)


Os exercícios práticos serão apresentados nos sistemas TerraView, SPRING, Gaia, PostgreSQL, ACCESS, GeoSERVER e uDig.


Material Didático

  • Apostila impressa para os exercícios práticos;
  • DVD com o material didático em PDF, os bancos de dados que serão manipulados no curso e a versão atual do SPRING.
    • Observação: Pode haver sobreposição de conteúdo entre os cursos por não haver pré-requisito para realizá-los.

Certificado

30 dias após o curso o aluno recebe o certificado da SELPER.
